# Webサイト大解剖

「正しいWebサイトとはなにか」をテーマにした、初心者向けWebサイト学習プロジェクト

## プロジェクト概要

このプロジェクトは、**初心者がWebサイトを作りたいと思ったときに、このサイトを見れば理解できて作れる！** というコンセプトで制作されました。

Webサイトの各要素（ヘッダー、ハンバーガーメニュー、カルーセル、検索ボックスなど）を分解し、インタラクティブなUIボタンとして配置。各ボタンをクリックすると、その要素の詳細な解説とCodePenでの実例を確認できます。

## コンセプト

- **わかりやすさ重視**: 初心者でも直感的に理解できるUI/UX設計
- **実践的な学習**: 実際のコード例をCodePenで確認しながら学べる
- **体系的な構成**: Webサイトの構成要素を網羅的に解説

## 主な機能・セクション

### Webサイトの構成要素

プロジェクトでは以下のWebサイト構成要素を詳しく解説しています：

1. **ヘッダー** ([header.html](header.html))
   - Webサイトの顔となるヘッダーの役割と実装方法
   - レスポンシブ対応のヘッダーデザイン

2. **ハンバーガーメニュー** ([hamburger.html](hamburger.html))
   - モバイルファーストデザインに必須のナビゲーション
   - アニメーション付きメニューの実装

3. **検索ボックス** ([search.html](search.html))
   - ユーザビリティを向上させる検索機能
   - デザインパターンと実装例

4. **カルーセル（スライダー）** ([slider.html](slider.html))
   - 魅力的なビジュアル表現のためのスライダー
   - レスポンシブ対応のカルーセル実装

5. **記事コンテンツ** ([article.html](article.html))
   - 読みやすい記事レイアウトの設計
   - コンテンツの構造化

6. **FAQ（よくある質問）** ([faq.html](faq.html))
   - アコーディオンUIの実装
   - ユーザーサポート向けデザイン

7. **問い合わせフォーム** ([query.html](query.html))
   - フォームのベストプラクティス
   - バリデーションとUX

8. **フッター** ([footer.html](footer.html))
   - サイト全体を締めくくるフッターデザイン
   - 必要な情報の配置方法

## 技術スタック

- **HTML5**: セマンティックなマークアップ
- **CSS3**: モダンなスタイリング、レスポンシブデザイン
- **JavaScript**: インタラクティブなUI実装
- **Font**: Noto Sans JP（Google Fonts）
- **Icons**: Font Awesome

## デザインの特徴

- **ダークテーマ**: 目に優しい暗色系のカラースキーム
- **レスポンシブデザイン**: モバイル・タブレット・デスクトップに対応
- **直感的なナビゲーション**: ロケットアイコンを使った視覚的な動線
- **インタラクティブなUI**: ホバーエフェクトやトランジションアニメーション

## 使い方

1. `top.html` をブラウザで開く
2. 学びたいWebサイト要素のボタンをクリック
3. 各要素の詳細ページで解説とコード例を確認
4. CodePenのリンクから実際の動作を確認
